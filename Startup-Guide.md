# General Information
 by executin `mkdocs serve` in the directory of `mkdocs.yml`.

[Official Documentation](https://www.mkdocs.org/user-guide/deploying-your-docs/)

## Quick Commands
`mkdocs serve` Start up the development server \
`mkdocs build` - Build the documentation site

## Project layout
```yml
mkdocs.yml      # The config file
docs/
    index.md    # The documentation homepage.
    ...         # Other markdown pages, images and other files
```
